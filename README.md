# WNN6231.github.io

> **个人博客 | Powered by Hexo**

这是我的个人博客仓库，依托 GitHub Pages 部署。网址为：[https://WNN6231.github.io](https://wnn6231.github.io/)

---

## 技术栈与特性
* **框架**: Hexo (Static Site Generator)
* **部署**: GitHub Pages + Actions (自动构建部署)
* **内容**: Markdown 编写，支持 LaTeX 数学公式渲染
* **功能**: 本地全站搜索、文章分类、标签云、摄影画廊

---

### 搭建这个博客的初衷

原本是想快速搭建个博客然后进行内容产出，但是翻阅各个框架和模板发现都没有自己喜欢的，这种模板个人自由度比较低，于是写到一半不满意便废弃了。
现在在开发基于NextJS框架的博客。

---

## 目录结构说明

| 文件夹/文件 | 内容描述 |
| :--- | :--- |
| **📁 2025** | **归档**：2025 年度所有文章的静态资源与索引 |
| **📁 about** | **关于我**：个人简历、技能树及联系方式页面 |
| **📁 photo** | **摄影**：存放生活照片及摄影作品集的静态页面 |
| **📁 categories / tags** | **分类索引**：全站文章的逻辑分类与关键词映射 |
| **📁 js / css** | **前端资源**：博客主题的自定义样式表与交互逻辑 |
| **📁 img** | **媒体库**：文章插图、封面图及相关视觉素材 |
| **📁 links** | **友链**：记录志同道合的小伙伴们的数字空间 |
| **📄 index.html** | **首页**：博客的入口展示页面 |
| **📄 local-search.xml** | **搜索索引**：支持全站内容即时检索的核心数据 |
| **📄 404.html** | **错误页**：当访问路径不存在时的引导页面 |

---

## 博客维护说明
1.  **主题优化**：定期在 `css` 文件夹中调整样式，保持视觉的一致性。
2.  **SEO 优化**：利用 `tags` 和 `categories` 提高搜索引擎对技术文章的收录。
3.  **安全性**：对于敏感的实验报告或私密信息，在发布前进行脱敏处理。
